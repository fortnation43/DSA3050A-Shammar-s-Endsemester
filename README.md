#  1. Project Overview
This project presents an advanced Power BI solution designed to transform raw sales data into meaningful business insights. The dashboard enables interactive analysis of sales performance across products, customers, regions, and time.

The solution includes data cleaning, data modelling, DAX calculations, and interactive visualisations to support data-driven decision-making.

---

#  2. Problem Statement

Businesses generate large volumes of sales data across products, customers, and regions, but often struggle to transform this raw data into actionable insights. Without proper analytics, it becomes difficult to identify key revenue drivers, understand customer behaviour, or detect performance imbalances across regions and product categories.

This project addresses this challenge by developing an interactive Power BI dashboard that enables stakeholders to analyse sales performance, profitability, and trends in order to support strategic decision-making.

---

#  3.This project was designed to answer the following key questions:

Which product categories generate the highest and lowest revenue?
How do sales and profit vary across different regions?
Which customer segment contributes the most to overall sales?
What are the trends in sales performance over time?
Which sub-categories and products are the top and bottom performers?
Is profitability growing at the same rate as sales?
How is revenue distributed geographically across regions?
Which products or segments should be prioritised for growth?
Are there signs of over-reliance on specific products or customers?
How can sales performance be improved through better targeting and resource allocation?

#  4. How the Dashboard Solves the Problem

The Power BI dashboard addresses the problem statement through the following:

### ✔ Data Transformation
Raw data was cleaned using Power Query by removing duplicates, handling missing values, correcting data types, and creating calculated columns.

### ✔ Data Modelling
A star schema was implemented linking a central FactSales table with dimension tables (Customer, Product, Geography, Date), enabling efficient filtering and analysis.

### ✔ DAX Calculations
Advanced DAX measures were created including:
- Total Sales
- Total Profit
- Profit Margin %
- Sales Growth %
- Year-to-Date Sales
- Distinct Customers
- Product Ranking

These measures enable deeper analytical insights beyond basic aggregation.

### ✔ Interactive Dashboard
The dashboard includes:
- KPI cards for high-level performance tracking  
- Trend analysis for sales over time  
- Category and sub-category breakdowns  
- Regional performance mapping  
- Drillthrough and tooltip pages for detailed exploration

# 📂 5. Dataset Description
The dataset contains sales transaction records including:

The dataset supports relational modelling and time-based analysis.
