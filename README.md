#  1. Project Overview
This project presents an advanced Power BI solution designed to transform raw sales data into meaningful business insights. The dashboard enables interactive analysis of sales performance across products, customers, regions, and time.

The solution includes data cleaning, data modelling, DAX calculations, and interactive visualisations to support data-driven decision-making.

---

#  2. Problem Statement
Organisations often struggle to extract meaningful insights from raw transactional data. The dataset contains large volumes of sales records that are not immediately useful for decision-making.

The challenge is to transform this raw data into a structured, interactive dashboard that helps stakeholders understand:
- sales performance trends
- customer behaviour
- product performance
- regional distribution
- profitability patterns

---

#  3. Objectives of the Project
The main objectives of this project are:

- To clean and transform raw sales data into a structured format suitable for analysis  
- To build a relational data model using fact and dimension tables  
- To develop DAX measures for advanced analytics such as profit margin, growth, and ranking  
- To design an interactive Power BI dashboard for business decision-making  
- To identify key insights, patterns, and trends in sales performance  
- To provide actionable recommendations based on data analysis  

---

#  4. How the Dashboard Solves the Problem

The Power BI dashboard addresses the problem statement through the following:

### ✔ Data Transformation
Raw data was cleaned using Power Query by removing duplicates, handling missing values, correcting data types, and creating calculated columns.

### ✔ Data Modelling
A star schema was implemented linking a central FactSales table with dimension tables (Customer, Product, Geography, Date), enabling efficient filtering and analysis.

### ✔ DAX Calculations
Advanced DAX measures were created including:
- Total Sales
- Total Profit
- Profit Margin %
- Sales Growth %
- Year-to-Date Sales
- Distinct Customers
- Product Ranking

These measures enable deeper analytical insights beyond basic aggregation.

### ✔ Interactive Dashboard
The dashboard includes:
- KPI cards for high-level performance tracking  
- Trend analysis for sales over time  
- Category and sub-category breakdowns  
- Regional performance mapping  
- Drillthrough and tooltip pages for detailed exploration
